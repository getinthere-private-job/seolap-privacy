서랍 (Seolap) 개인정보 처리방침

시행일: 2026년 4월 3일

본 개인정보 처리방침은 서랍(Seolap) 앱(이하 "앱")의 개인정보 수집, 이용, 보관 및 보호에 관한 사항을 안내합니다.

1. 수집하는 개인정보

서랍 앱은 별도의 회원가입이나 계정 생성을 요구하지 않으며, 사용자의 개인정보를 직접 수집하지 않습니다. 모든 콘텐츠(서랍, 카드, 사진, 텍스트, 음성, PDF)는 사용자의 기기에만 저장됩니다.

2. 데이터 저장 및 보관

앱에서 생성하는 모든 콘텐츠는 기기의 로컬 저장소(SQLite 데이터베이스)에 저장됩니다. 사용자의 데이터는 별도의 서버로 전송되거나 클라우드에 백업되지 않습니다.

앱을 삭제하면 모든 데이터가 기기에서 영구 삭제됩니다.

3. 기기 권한

앱은 다음 기기 권한을 요청할 수 있습니다:

권한용도데이터 전송
카메라카드에 사진 촬영없음 (기기 저장)
사진 라이브러리기존 사진 선택 및 추가없음 (기기 저장)
마이크음성 메모 녹음AI 요약 사용 시에만 전송 (아래 참조)

모든 권한은 선택적이며, 사용자가 거부해도 해당 기능 외의 앱 사용에는 영향이 없습니다.

4. 위치 정보

앱은 사용자의 실시간 위치를 추적하지 않습니다. 사진에 포함된 GPS 메타데이터(EXIF)를 읽어 카카오맵에 마커로 표시합니다. 위치 데이터는 기기에서만 처리되며 외부로 전송되지 않습니다.

5. AI 음성 요약 기능

사용자가 AI 음성 요약 기능을 사용할 경우, 녹음된 음성 데이터가 Google Gemini API로 전송되어 텍스트 요약이 생성됩니다.

  • 전송 대상: Google Gemini 2.5 Flash API
  • 전송 데이터: 음성 녹음 파일
  • 목적: 음성 내용의 텍스트 요약 생성
  • 보관: 요약 결과는 기기에 저장되며, Google 측의 데이터 처리는 Google Gemini API 이용약관을 따릅니다.

AI 요약 기능은 사용자가 명시적으로 요청할 때만 작동하며, 자동으로 음성 데이터를 전송하지 않습니다.

6. 광고

서랍 앱은 Google AdMob을 통해 리워드 광고 및 네이티브 광고를 제공합니다. AdMob은 광고 제공 및 분석을 위해 다음 정보를 수집할 수 있습니다:

  • 기기 식별자 (광고 ID)
  • IP 주소
  • 앱 사용 데이터

iOS 14 이상에서는 앱 추적 투명성(ATT) 동의를 요청하며, 사용자가 거부할 경우 맞춤 광고가 제한됩니다. 자세한 내용은 Google 개인정보 처리방침을 참조하세요.

8. PDF 공유

서랍의 카드를 PDF로 내보내거나 ZIP으로 공유할 때, 파일은 기기에서 생성되며 사용자가 직접 수신자를 선택합니다. 앱의 서버를 경유하지 않습니다.

9. 어린이 개인정보

서랍 앱은 만 13세 미만의 어린이를 대상으로 하지 않으며, 의도적으로 어린이의 개인정보를 수집하지 않습니다. 만 13세 미만 어린이의 정보가 수집된 것을 알게 될 경우, 해당 정보를 즉시 삭제합니다.

10. 처리방침 변경

본 처리방침이 변경될 경우 앱 내 공지 또는 본 페이지를 통해 사전 안내합니다. 변경된 처리방침은 게시한 날로부터 효력이 발생합니다.
